# wasgehtFL — Content Plan
**Monat 1 · April/Mai 2026**

---

## Strategie-Übersicht

**Ziel Monat 1:** Konsistenz aufbauen, Reichweite durch Venue-Tagging, ersten viralen Post landen
**Posting-Frequenz:** 1 Reel/Tag + Daily Story + WhatsApp Push
**Beste Posting-Zeit:** Reels 17:00–19:00 Uhr / Stories 08:00 + 17:00 Uhr

## Reel-Skripte (Templates)

### Template A — Daily Events Reel
```
[Hook, 0–2s]
Text-Overlay: "Was geht HEUTE in Flensburg? 👇"
Musik: trending, energetisch

[Event 1, 2–4s]
Venue-Foto + Text: "[Eventname]"
Sub-Text: "[Uhrzeit] · [Ort]"

[Event 2, 4–6s]
[gleiche Struktur]

[Event 3, 6–8s]
[gleiche Struktur]

[Outro, 8–10s]
"Mehr Events täglich → @wasgehtFL"
Logo + Coral Background
```

### Template B — Geheimtipp Reel
```
[Hook, 0–3s]
Text: "Diesen Spot in Flensburg kennen die meisten nicht."

[Reveal, 3–8s]
Venue-Foto/Video
Langsamer, cinematic

[Info, 8–12s]
Name + Was es ist + Warum es sich lohnt

[CTA, 12–15s]
"Geht hin. Ihr könnt euch bei uns bedanken. 🧡"
```

### Template C — Wochenend-Special
```
[Hook, 0–2s]
"Dein Wochenende in Flensburg. Hier sind die Top 5."

[Events 1–5, je 2–3s]
Nummeriert: "01 / 02 / 03..."
Coral-Overlay mit weißem Text

[Outro, 14–16s]
"Welches ist euer Favorit? Kommentiert 👇"
"Folgt für täglich neue Events"
```
